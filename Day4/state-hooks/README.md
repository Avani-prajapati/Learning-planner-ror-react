# Sweet Shop App

A simple React application where users can add sweets and view them in a list.

The app allows users to open a form, enter sweet details such as name, price, and description, and add the sweet to the list. Once a sweet is added successfully, a toast notification is displayed to confirm the action. Error messages are also shown if the form fields are not filled correctly.

When the application loads, a welcome toast message is displayed using `useEffect`. The app also demonstrates conditional rendering by showing or hiding the form based on user interaction.

Each sweet is displayed in a card-style layout with its name, description, and price.

This project demonstrates a simple interactive UI with form handling and user feedback using toast notifications.
